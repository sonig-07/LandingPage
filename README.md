# React Landing Page (Next.js)

## Assignment Overview

This project is a simple landing page built using **React.js with the Next.js framework**. The main objective of this project is to demonstrate understanding of React, component structure, basic user interaction, and deployment.

The UI is intentionally kept clean and minimal to focus on functionality, readability, and correct setup.

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/sonig-07/LandingPage
```

2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
npm run dev
```

4. Open browser:

```
http://localhost:3000
```
